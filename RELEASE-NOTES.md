# ddth-mappings release notes

## v0.1.0 - 2017-08-25

First release:

- 3 mapping types:
  - One-One (OO) mappings
  - Many-One (MO) mappings
  - Many-Many (MM) mappings
- data types:
  - `Byte`, `Short`, `Integer`, `Long`, `String`
- 2 storage types:
  - In-memory: classes `InmemMapping*Dao`
  - Cassandra: classes `CqlMapping*Dao`
